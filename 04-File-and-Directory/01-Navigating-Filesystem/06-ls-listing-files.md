

# Listing Files 

`ls [OPTIONS] [FILE]`


* **`ls -l`** → Lists files in **long detailed format**
* **`ls -a`** → Show all, including **hidden**.
* **`ls -d`** → Show **directory** only.
* **`ls -ld`** → **Detailed** directory info.
* **`ls -r`** → Lists files in **reverse alphabetical order**
* **`ls -t`** → Lists files **sorted by latest time first**
* **`ls -lt`** → Lists files in **long format**, sorted by **latest time first**
* **`ls -lrt`** → **Sort** by time (old → new).
* **`ls -S`** → Lists files **sorted by largest size first**
* **`ls -l -S`** → Lists files in **long format**, sorted by **largest size first**
* **`ls -lSr`** → Lists files in **long format**, sorted by **smallest size first** (reverse of `-S`) table format
