#  Configuring Network Devices 

#### Key Questions

* **Wired or Wireless?**

  * Wireless setup includes extra settings like **security**.
  * Wired setup is usually simpler.

* **DHCP or Static IP?**

  * **DHCP**: Network settings assigned automatically.
  * **Static IP**: Network settings set manually.


### Common Usage

| Device Type | Network Type | IP Assignment  |
| ----------- | ------------ | -------------- |
| Desktop     | Wired        | Static or DHCP |
| Laptop      | Wireless     | DHCP           |

---


>* **Wired devices** often use static IPs but can use DHCP.
>* **Wireless devices** almost always use DHCP due to mobility.
