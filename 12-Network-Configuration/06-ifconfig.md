

## **ifconfig — View Network Settings**

The `ifconfig` (interface configuration) command shows network interface details.

Syntax : `ifconfig [OPTIONS]`

**Note:** `iwconfig` is similar but used **only for wireless interfaces**.


>1. **`ifconfig`** – Display network configuration information
>2. **`ip addr show`** – Modern replacement for `ifconfig`

>`ifconfig` can temporarily change network settings, but permanent changes should be done using **network configuration tools**, not ifconfig.
